Технические характеристики

Favicon (для веб-приложения):

  • favicon.ico - 16x16, 32x32 (multi-size ICO)
  • favicon-16x16.png - для старых браузеров
  • favicon-32x32.png - стандартный размер
  • apple-touch-icon.png - 180x180 (iOS Safari)

Mobile App Icon:

  • icon-192x192.png - Android
  • icon-512x512.png - Android (high-res)
  • icon-180x180.png - iOS
  • icon-1024x1024.png - App Store/Play Store

Логотип:

  • logo.svg - векторный формат (предпочтительно)
  • logo.png - 300x100 (для использования в документах)
  • logo-white.svg - для тёмных фонов
  • logo-icon-only.svg - только иконка без текста
