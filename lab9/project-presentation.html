📋 Обзор проекта

🎯 Цель

Разработать современный интернет-магазин электроники с полным функционалом для покупателей и администраторов

👥 Целевая аудитория

  • Энтузиасты технологий (25-35 лет)
  • Профессионалы (30-45 лет)
  • Студенты (18-25 лет)
  • Бизнес-клиенты

✨ Ключевые функции

  • Каталог с фильтрацией
  • Корзина и заказы
  • Авторизация (JWT)
  • Отзывы и рейтинги
  • Личный кабинет
  • Админ-панель

🏗️ Архитектура

Backend: Node.js + Express + PostgreSQL

Frontend: React + Vite + Context API

Deploy: Render + Vercel

CI/CD: GitHub Actions
